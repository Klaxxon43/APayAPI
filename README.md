# APayAPI
ШАБЛОН ДЛЯ ИНТЕГРАЦИИ APay (@apays_bot) В БОТА

Для получения токена напишите в бота @apays_bot в телеграмме, оставьте заявку, написав туда название проекта и оставив ссылку на него. Спустя некоторое время его вероятно примут и вы получите SECRET_KEY и CLIENT_ID (ID проекта). Добавьте их в свой файл config.
